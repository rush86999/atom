## ğŸš€ Quick Start

### 1. Clone & Setup
```bash
git clone https://github.com/rush86999/atom.git
cd atom
```

### 2. Start Backend
```bash
cd backend
python main_api_app.py
# or use: python start_simple_backend.py
```

### 3. Start Frontend
```bash

```
