## üöÄ Quick Start

### Enterprise Deployment
```bash
# Clone enterprise platform
git clone https://github.com/rush86999/atom.git
cd atom

# Start all services
docker-compose up -d

# Access platform
open http://localhost:3000
```

### Development Setup
```bash
# Frontend development
cd frontend-nextjs
npm install
npm run dev

# Backend development
cd backend
pip install -r requirements.txt
python main_api_app.py

# Desktop development
cd desktop/tauri
npm install
npm run tauri dev
```

### Service Integration
```bash
# Configure integrations
cp .env.example .env
# Edit .env with your API keys

# Start integration services
npm run start:integrations
```
