## üí¨ Basic Chat Commands

### Getting Help
- `help` - Show available commands
- `what can you do?` - List capabilities
- `status` - Check system and service status

### Service Management
- `connect [service]` - Connect to a service (e.g., "connect Slack")
- `disconnect [service]` - Disconnect from a service
- `list services` - Show available integrations

### Task Management
- `create task [description]` - Create a new task
- `show my tasks` - Display current tasks
- `complete task [task name]` - Mark task as complete

## üîç Troubleshooting

### Common Issues

#### Service Connection Problems
**Problem**: "Cannot connect to [service]"
**Solution**:
1. Check internet connection
2. Verify service credentials
3. Re-authenticate through OAuth
4. Check if service API is operational

#### Command Not Working
**Problem**: "Command not recognized"
**Solution**:
1. Use the `help` command to see available options
2. Check service connectivity status
3. Try rephrasing the command
4. Ensure proper service name spelling

#### Performance Issues
**Problem**: Slow response times
**Solution**:
1. Check system resources
2. Verify AI provider connectivity
3. Clear chat history if needed
4. Restart the application

### Advanced Troubleshooting

#### Log Files
- Access logs through Settings ‚Üí Advanced ‚Üí View Logs
- Share error messages with support team
- Check for specific service error codes

#### Network Diagnostics
- Test individual service APIs
- Verify firewall and proxy settings
- Check DNS resolution for service domains
