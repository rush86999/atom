## üé≠ Understanding Sentiment Analysis

### How It Works
The system analyzes your messages to understand your emotional state and responds appropriately.

### Examples

**Positive Sentiment:**
```
You: "This feature is amazing! Great work team!"
Atom: "Great to hear you're enjoying this feature! Here's some additional information that might help."
```

**Negative Sentiment:**
```
You: "This is so frustrating, nothing works properly"
Atom: "I understand you're frustrated. Let me help you resolve these issues step by step."
```

**Neutral Sentiment:**
```
You: "Can you show me how to update my profile?"
Atom: "Sure, I can help you update your profile. Here are the steps..."
```

## üîç Entity Extraction

### What Gets Detected
The system automatically identifies and processes:
- **Email addresses**: user@example.com
- **Phone numbers**: +1-555-0123
- **URLs**: https://example.com
- **Financial data**: $ amounts, percentages
- **Dates**: 2024-01-01, tomorrow, next week

### Example Usage
```
You: "My email is john@company.com and I need help with a $500 invoice"
Atom: "I'll help you with the invoice for $500. I've noted your email john@company.com for follow-up."
```

## üéØ Intent Recognition

### Common Intents Detected

| Intent Type | Examples | System Response |
|-------------|----------|-----------------|
| **Help Request** | "I need help", "Can you assist" | Detailed assistance with step-by-step guidance |
| **Complaint** | "This isn't working", "I'm having issues" | Empathetic response with troubleshooting |
| **Positive Feedback** | "Great job", "This is amazing" | Appreciative response with additional value |
| **Information Request** | "What is", "How do I" | Clear, informative answers |
| **Action Request** | "Can you", "Please do" | Direct action with confirmation |

### Example
```
You: "I need help setting up my calendar integration"
Atom: "I can help you set up calendar integration. Let me guide you through the process..."
```

## üîÑ When to Use Enhanced vs Standard Chat

### Use Enhanced Chat When:
- You need personalized, context-aware responses
- You're expressing emotions or frustrations
- You want proactive suggestions
- You're working on complex, multi-step tasks

### Use Standard Chat When:
- You need quick, straightforward answers
- You're performing simple queries
- System performance is critical
