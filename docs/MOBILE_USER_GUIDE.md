## Getting Started

### Download and Install

#### iOS

1. Open the App Store on your iPhone or iPad
2. Search for "Atom AI"
3. Tap "Get" to download and install
4. Launch the app from your home screen

#### Android

1. Open Google Play Store on your Android device
2. Search for "Atom AI"
3. Tap "Install" to download and install
4. Launch the app from your app drawer

### First Launch

When you first open Atom Mobile:

1. **Welcome Screen**: Swipe through the introduction screens to learn about key features
2. **Sign In**: Sign in with your existing Atom account, or create a new account
3. **Permissions**: Grant necessary permissions (location, notifications, camera, etc.)
4. **Complete Setup**: You're ready to start using Atom!

## Authentication

### Creating an Account

1. Tap "Get Started" on the welcome screen
2. Enter your email address
3. Create a secure password (minimum 8 characters)
4. Verify your email address using the link sent to your inbox
5. Complete your profile (optional)

### Signing In

#### Email and Password

1. Enter your registered email address
2. Enter your password
3. Tap "Sign In"

#### Biometric Authentication (Face ID / Touch ID)

For quick access, enable biometric authentication:

1. Go to **Settings** > **Security**
2. Enable "Biometric Login"
3. Authenticate with your face or fingerprint
4. Next time, just use Face ID/Touch ID to sign in

#### OAuth Providers

You can also sign in with:

- **Google**: Tap "Sign in with Google"
- **Apple** (iOS only): Tap "Sign in with Apple"

### Signing Out

To sign out:

1. Go to **Settings** (tab bar at the bottom)
2. Scroll down and tap "Sign Out"
3. Confirm by tapping "Yes, Sign Out"

**Note**: Signing out clears your offline data. You'll need to sign in again to access your account.

## Chat with Agents

### What Are Agents?

Agents are AI-powered assistants that help you automate tasks, answer questions, and execute workflows. Each agent has different capabilities based on its maturity level and training.

### Accessing Agent Chat

1. Tap the **Chat** tab in the bottom navigation
2. Tap **"Select an Agent"** at the top
3. Browse available agents or search by name
4. Tap on an agent to start chatting

### Chat Interface

The chat interface consists of:

- **Agent Avatar**: Shows the selected agent's avatar and name
- **Chat Messages**: Conversation history with the agent
- **Message Input**: Type your message or use voice input
- **Send Button**: Tap to send your message

### Sending Messages

#### Text Messages

1. Tap the message input field at the bottom
2. Type your question or request
3. Tap the send button (paper plane icon)

#### Voice Messages

1. Tap the microphone icon next to the input field
2. Speak your message
3. Tap "Send" to transcribe and send

#### Attachments

1. Tap the attachment icon (paperclip)
2. Choose:
   - **Photo**: Take or select a photo
   - **Document**: Select a file from your device
3. Add a message (optional)
4. Tap send

### Agent Capabilities

Different agents can perform different actions:

#### Student Agents
- Present information (charts, markdown)
- Answer questions from existing data
- No execution permissions

#### Intern Agents
- Stream responses in real-time
- Present forms for data collection
- Require approval for actions

#### Supervised Agents
- Execute workflows with real-time monitoring
- Submit forms and make state changes
- Supervised by advanced agents

#### Autonomous Agents
- Full autonomy to execute actions
- No approval required
- Full access to all capabilities

### Viewing Agent Details

To view an agent's capabilities and maturity level:

1. Tap the agent avatar at the top of the chat
2. View:
   - **Maturity Level**: Student, Intern, Supervised, or Autonomous
   - **Capabilities**: What the agent can do
   - **Confidence Score**: How confident the agent is in its abilities
   - **Statistics**: Execution count, success rate, etc.

### Chat History

Your conversations are automatically saved. To access previous chats:

1. Tap the back arrow in the chat header
2. Scroll through the list of conversations
3. Tap on any conversation to resume

## Workflows

### What Are Workflows?

Workflows are automated sequences of actions that accomplish specific tasks. You can trigger workflows, monitor their execution, and view logs.

### Accessing Workflows

1. Tap the **Workflows** tab in the bottom navigation
2. Browse your available workflows

### Triggering a Workflow

1. Tap on a workflow card
2. Review the workflow details:
   - Description
   - Required inputs
   - Estimated duration
3. Fill in any required input fields
4. Tap **"Start Workflow"**

### Monitoring Execution

While a workflow is running:

1. You'll see a **progress indicator** showing the current step
2. Real-time **status updates** as the workflow progresses
3. **Execution logs** showing detailed information

#### Execution States

- **Pending**: Workflow is queued
- **Running**: Workflow is currently executing
- **Completed**: Workflow finished successfully
- **Failed**: Workflow encountered an error

### Viewing Execution Logs

1. Tap on a completed or failed execution
2. View detailed logs including:
   - Start and end time
   - Each step's status
   - Error messages (if failed)
   - Output data

### Re-running Workflows

1. Tap on a workflow execution
2. Tap **"Re-run"** in the top right
3. Modify inputs if needed
4. Tap **"Start"** to run again

## Canvas Presentations

### What Are Canvases?

Canvases are visual presentations created by agents to display information, collect data, or guide you through processes.

### Canvas Types

#### 1. Generic Canvas
Free-form content with custom layout

#### 2. Documentation Canvas
Structured documentation with sections and formatting

#### 3. Email Canvas
Email composition and preview

#### 4. Sheets Canvas
Spreadsheet-style data presentation

#### 5. Orchestration Canvas
Workflow orchestration with steps and dependencies

#### 6. Terminal Canvas
Terminal output and command execution

#### 7. Coding Canvas
Code display and execution

### Viewing a Canvas

When an agent presents a canvas:

1. The canvas opens in a full-screen modal
2. Interact with the content:
   - **Scroll**: Swipe up/down to view more content
   - **Forms**: Fill in input fields
   - **Buttons**: Tap to submit or take actions
3. Tap **"Close"** when done

### Canvas Actions

#### Submitting Forms

1. Fill in all required fields (marked with *)
2. Tap **"Submit"**
3. Wait for confirmation
4. The agent will process your submission

#### Interacting with Charts

- **Line Charts**: Tap data points to see values
- **Bar Charts**: Tap bars for details
- **Pie Charts**: Tap segments for percentages
- **Zoom**: Pinch to zoom in/out (supported charts)

#### Saving Canvas Data

1. Tap the **share icon** in the canvas header
2. Choose:
   - **Save as Image**: Save canvas screenshot
   - **Export Data**: Export underlying data
   - **Copy to Clipboard**: Copy text or data

## Analytics Dashboard

### Accessing Analytics

1. Tap the **Analytics** tab in the bottom navigation

### Dashboard Overview

The analytics dashboard shows:

#### Key Metrics
- **Total Executions**: Number of workflow executions
- **Success Rate**: Percentage of successful executions
- **Average Duration**: Average execution time
- **Active Agents**: Number of agents you've used

#### Performance Charts
- **Execution Trend**: Line chart showing executions over time
- **Agent Usage**: Bar chart showing most-used agents
- **Workflow Performance**: Success rate by workflow

### Filtering Data

1. Tap the **filter icon** in the top right
2. Select:
   - **Date Range**: Last 7 days, 30 days, 90 days, or custom
   - **Agent**: Filter by specific agent
   - **Workflow**: Filter by specific workflow
3. Tap **"Apply"**

### Exporting Analytics

1. Tap the **export icon** in the top right
2. Choose format:
   - **CSV**: For spreadsheet analysis
   - **JSON**: For data processing
   - **PDF**: For reports
3. Select destination:
   - **Email**: Send via email
   - **Save**: Download to device

## Offline Mode

### How Offline Mode Works

Atom Mobile supports offline access to:

- Agent conversations
- Workflow definitions
- Canvas presentations
- Execution history

### Syncing for Offline Use

Data is automatically synced when:

1. You're connected to Wi-Fi (if enabled)
2. The app is open in the foreground
3. Sync runs every 15 minutes

### Using Offline Features

#### Chat with Agents

1. View previous conversations (already synced)
2. Compose new messages (queued until online)
3. View cached agent responses

#### Trigger Workflows

1. View workflow definitions (synced)
2. Start workflows (queued until online)
3. Check execution status (cached)

#### View Canvases

1. View previously opened canvases (cached)
2. Interact with form data (queued until online)

### Offline Indicator

When you're offline:

- A **banner** appears at the top showing "Offline Mode"
- **Pending actions** count shows queued items
- Tap **"Sync Now"** to manually sync when back online

### Managing Offline Storage

#### Check Storage

1. Go to **Settings** > **Offline Mode**
2. View **"Storage Used"**

#### Clear Offline Cache

1. Tap **"Clear Offline Cache"**
2. Confirm by tapping **"Clear"**
3. All offline data will be removed

#### Storage Warning

When offline storage reaches 80%:

- A warning appears in settings
- Old data is automatically removed (LRU eviction)
- You can manually clear cache to free space

## Troubleshooting

### App Won't Launch

**Problem**: App crashes on startup

**Solutions**:
1. Restart your device
2. Update to the latest app version
3. Clear app cache (Settings > Clear Data > Offline Cache)
4. Reinstall the app (last resort)

### Can't Sign In

**Problem**: Login fails

**Solutions**:
1. Check internet connection
2. Verify email and password are correct
3. Reset password if forgotten
4. Check if account is locked (contact support)

### Notifications Not Working

**Problem**: Not receiving push notifications

**Solutions**:
1. Check notification permissions (Settings > Notifications > Atom Mobile)
2. Enable notifications in app (Settings > Notifications)
3. Check Do Not Disturb mode
4. Ensure battery optimization is disabled

### Workflows Failing

**Problem**: Workflows fail to execute

**Solutions**:
1. Check internet connection
2. Verify all required inputs are provided
3. Check workflow logs for error details
4. Try re-running the workflow
5. Contact support if issue persists

### Offline Sync Issues

**Problem**: Offline data won't sync

**Solutions**:
1. Check internet connection
2. Pull to refresh in any tab
3. Manually trigger sync (tap offline banner > Sync Now)
4. Clear offline cache and re-sync

### App Running Slow

**Problem**: App is sluggish or unresponsive

**Solutions**:
1. Close other apps running in background
2. Clear offline cache (Settings > Offline Mode > Clear Cache)
3. Check available device storage
4. Restart the app
5. Restart your device

### Battery Drain

**Problem**: App draining battery quickly

**Solutions**:
1. Disable background refresh (iOS Settings > General > Background App Refresh)
2. Reduce sync frequency (Settings > Offline Mode)
3. Disable location services (if not needed)
4. Enable battery saver mode

## FAQ

### General

**Q: Is Atom Mobile free?**
A: Yes, Atom Mobile is free to download and use. Some features may require a subscription in the future.

**Q: What devices are supported?**
A: iOS 13+ (iPhone and iPad) and Android 8+.

**Q: Can I use Atom Mobile on multiple devices?**
A: Yes, sign in with the same account on all your devices.

### Account

**Q: How do I reset my password?**
A: Tap "Forgot Password" on the sign-in screen and follow the instructions.

**Q: Can I change my email address?**
A: Currently, you'll need to contact support to change your email.

### Features

**Q: What's the difference between agent maturity levels?**
A:
- **Student**: View-only, no execution
- **Intern**: Requires approval for actions
- **Supervised**: Monitored execution
- **Autonomous**: Full independence

**Q: Can I create custom workflows?**
A: Currently, workflows are managed in the web app. Mobile app support is coming soon.

**Q: How long are chat histories stored?**
A: Chat histories are stored indefinitely unless you delete them.

### Privacy and Security

**Q: Is my data encrypted?**
A: Yes, all data is encrypted in transit and at rest.

**Q: Does Atom Mobile sell my data?**
A: No, we do not sell your personal data. See our Privacy Policy for details.

**Q: Can I use biometric authentication?**
A: Yes, Face ID (iOS) and fingerprint (Android) are supported.

### Offline Mode

**Q: How much data is cached offline?**
A: Up to 50MB by default, adjustable in settings.

**Q: Can I use the app completely offline?**
A: You can view synced data and compose messages, but actions require internet.

### Troubleshooting

**Q: How do I report a bug?**
A: Go to Settings > Help & Support > Report a Bug.

**Q: How do I contact support?**
A: Email support@atom-platform.com or use in-app support (Settings > Help & Support).
