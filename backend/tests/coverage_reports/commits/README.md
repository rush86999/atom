# Per-Commit Coverage Reports

This directory stores coverage reports for each CI run.

## Format

Filename: `{short_hash}_coverage.json`

Content:
```json
{
  "commit": "abc123def456...",
  "short_hash": "abc123de",
  "timestamp": "2026-03-01T12:34:56Z",
  "overall_coverage": 65.43,
  "covered_lines": 12345,
  "total_lines": 18876,
  "branch_coverage": 58.32,
  "module_breakdown": {
    "core": {"percent": 62.1, "covered": 5000, "total": 8050},
    "api": {"percent": 71.5, "covered": 4000, "total": 5594},
    "tools": {"percent": 58.2, "covered": 3345, "total": 5747}
  },
  "files_below_80": 45,
  "top_uncovered_files": [
    {"path": "core/workflow_engine.py", "uncovered": 543, "total": 1089, "percent": 50.1},
    ...
  ]
}
```

## Retention

Reports are retained for 90 days by default. Old reports are automatically cleaned up by CI.

## Usage

### List reports

```bash
python3 backend/tests/scripts/per_commit_report_generator.py --list
```

### Generate report manually

```bash
python3 backend/tests/scripts/per_commit_report_generator.py \
  --coverage-file backend/tests/coverage_reports/metrics/coverage.json \
  --commits-dir backend/tests/coverage_reports/commits
```

### Cleanup old reports

```bash
python3 backend/tests/scripts/per_commit_report_generator.py \
  --commits-dir backend/tests/coverage_reports/commits \
  --cleanup \
  --retention-days 90
```

## Report Fields

- `commit`: Full git commit hash
- `short_hash`: First 8 characters of commit hash
- `timestamp`: UTC timestamp in ISO 8601 format
- `overall_coverage`: Line coverage percentage
- `covered_lines`: Number of covered lines
- `total_lines`: Total number of lines
- `branch_coverage`: Branch coverage percentage
- `module_breakdown`: Coverage per module (core, api, tools, skills)
- `files_below_80`: Count of files with coverage < 80%
- `top_uncovered_files`: Top 10 files with most uncovered lines

## CI Integration

Reports are automatically generated by `.github/workflows/coverage-report.yml` after each test run and committed to this directory.
