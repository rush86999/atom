## Problem Statement

The last remaining collection error during full test suite:

```
ERROR collecting tests/test_excel_export_analytics.py
AttributeError: module 'numpy' has no attribute 'short'
```

### Root Cause Analysis (Discovered During Investigation)

1. **test_cognitive_tier_e2e.py** imports `main_api_app_safe.py` at module level (line 35)
2. **main_api_app_safe.py** creates an incomplete mock of numpy:
   ```python
   np_mock = mock_package("numpy")  # Creates empty module without .short
   sys.modules["numpy.core"] = MagicMock()
   sys.modules["numpy._core"] = MagicMock()
   sys.modules["numpy.core.multiarray"] = MagicMock()
   sys.modules["numpy._core.multiarray"] = MagicMock()
   ```
3. **test_excel_export_analytics.py** imports `discord_analytics_engine` at module level
4. **discord_analytics_engine.py** imports `openpyxl` which tries to access `numpy.short`
5. The mock numpy doesn't have `.short`, causing `AttributeError`

**Binary Search Results:**
- test_a*.py + test_b*.py + test_excel_export_analytics.py: 646 tests, 0 errors
- test_c*.py + test_excel_export_analytics.py: 1,218 tests, 1 error

- Root cause: `main_api_app_safe.py` imported by `test_cognitive_tier_e2e.py`

### Why Existing Cleanup Doesn't Work

The `pytest_collection` hook in conftest.py only runs once at the START of collection, not between each test file import during collection.

## Solution Implemented

### Module-Level Cleanup (Primary Fix)

Added cleanup code at the top of `test_excel_export_analytics.py`:
1. Detects mocked/incomplete numpy modules (checks for missing `.short` attribute)
2. Removes mocked module AND all its submodules (`numpy.core.*`, `numpy._core.*`, etc.)
3. Removes openpyxl modules that may have cached references to corrupted numpy
4. Forces clean import of real numpy when discord_analytics_engine is imported

```python
def _is_mocked_module(module, mod_name):
    """Check if a module is mocked or incomplete."""
    if module is None:
        return True
    if hasattr(module, '_spec_class'):  # MagicMock detection
        return True
    if mod_name == "numpy" and not hasattr(module, 'short'):  # Incomplete numpy
        return True
    return False

for mod in ["numpy", "pandas", "lancedb", "pyarrow"]:
    if mod in sys.modules and _is_mocked_module(sys.modules[mod], mod):
        # Remove the module and ALL submodules
        keys_to_remove = [k for k in sys.modules.keys() if k == mod or k.startswith(mod + '.')]
        for key in keys_to_remove:
            sys.modules.pop(key, None)
        # Also remove openpyxl modules with cached references
        for openpyxl_mod in list(sys.modules.keys()):
            if openpyxl_mod.startswith('openpyxl'):
                sys.modules.pop(openpyxl_mod, None)
```

def pytest_collection_modifyitems(session, config, items):
