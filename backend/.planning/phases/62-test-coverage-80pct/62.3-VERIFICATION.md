### What Doesn't Work
- Full test suite collection still has 1 error: `test_excel_export_analytics.py`
- Error: `AttributeError: module 'numpy' has no attribute 'short'`
- Happens during openpyxl import in discord_analytics_engine.py

### Root Cause Analysis
The error is complex:
1. During full suite collection, some file imported AFTER test_browser_agent_ai.py causes numpy to be in a broken state
2. When test_excel_export_analytics.py is later collected, openpyxl tries to access `numpy.short` which fails
3. The issue only occurs with the full test suite, not subsets
