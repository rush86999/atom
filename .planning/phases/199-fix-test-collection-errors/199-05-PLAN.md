<task type="auto">
  <name>Calculate Impact Scores for Target Modules</name>
  <files>.planning/phases/199-fix-test-collection-errors/199-05-TARGETS.md</files>
  <action>
    For each candidate module, calculate impact score:

    Impact Score = (Lines × Coverage Gap) / Estimated Effort

    Where:
    - Lines = total lines in module
    - Coverage Gap = 85 - current_coverage (for modules <85%)
    - Estimated Effort:
      * Simple logic: 1 test per 20 lines
      * Medium complexity: 1 test per 30 lines
      * High complexity: 1 test per 40 lines

    Example calculations:
    1. agent_governance_service (400 lines, 62% coverage):
       Gap = 23% = 92 lines
       Effort = 400/30 = 13 tests (medium complexity)
       Impact = (400 × 0.23) / 13 = 7.1 lines/test

    2. trigger_interceptor (250 lines, 74% coverage):
       Gap = 11% = 27.5 lines
       Effort = 250/30 = 8 tests
       Impact = (250 × 0.11) / 8 = 3.4 lines/test

    3. student_training_service (400 lines, blocked):
       Gap = unknown (unblock first)
       Effort = 15-20 tests (schema fixes)
       Impact = unknown until unblocked

    Rank by impact score (higher = better ROI for test effort).
  </action>
  <verify>
    Run: cat .planning/phases/199-fix-test-collection-errors/199-05-TARGETS.md | grep "Impact Score"
    Expected: 5-10 modules ranked by impact score
  </verify>
  <done>
    Impact scores calculated for all candidate modules
    Modules ranked by ROI (lines covered per test)
    High-impact targets (score >5) identified
