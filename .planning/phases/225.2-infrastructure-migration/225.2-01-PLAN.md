        # Import BYOKHandler for actual LLM integration
        from core.llm.byok_handler import BYOKHandler

        handler = BYOKHandler()
        response = await handler.chat_completion(
            provider="anthropic",  # Using default
            model="claude-3-5-sonnet-20241022",

            temperature=0.3

3. Replace method call: `handler.chat_completion()` → `service.generate()`
4. Update parameters:
   - `provider="anthropic"` → LLMService auto-selects
   - `model="claude-3-5-sonnet-20241022"` → Let LLMService use default or specify
   - `messages` → Convert to prompt string
5. Preserve response parsing logic in `_parse_llm_response()`
