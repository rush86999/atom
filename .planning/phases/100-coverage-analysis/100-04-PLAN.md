<task type="auto">
  <name>Create coverage trend tracker script</name>
  <files>backend/tests/scripts/coverage_trend_tracker.py</files>
  <action>
    Create Python script with trend tracking functionality:

    1. **record_snapshot(coverage_data, commit_hash=None)** function:
       - Extract overall coverage: percent_covered, covered_lines, total_lines
       - Extract module breakdown: core, api, tools percentages
       - Get current timestamp (UTC)
       - Get commit hash from git or accept parameter
       - Return snapshot dict

    2. **get_trend_history()** function:
       - Load coverage_trend_v5.0.json (create if doesn't exist)
       - Initialize structure: {baseline, history: [], current, metadata}
       - Return existing data or new structure

    3. **calculate_delta(current, previous)** function:
       - Calculate absolute change: current - previous
       - Calculate relative change: (current - previous) / previous * 100
       - Return {absolute_change, relative_change, direction}

    4. **update_trend_data(snapshot, trend_data)** function:
       - Append snapshot to history array
       - Keep only last 30 entries (prevent unbounded growth)
       - Update current field
       - Set baseline if not set (first snapshot becomes baseline)
       - Return updated trend_data

    5. **write_trend_data(trend_data)** function:
       - Save to metrics/coverage_trend_v5.0.json
       - Create daily snapshot in trends/YYYY-MM-DD_coverage_trend.json

    6. **generate_visualization(trend_data)** function:
       - Create ASCII chart showing coverage over time
       - Mark 80% target line
       - Show baseline and current values
       - Return formatted string for console output

    7. **main()** orchestration:
       - Parse args: --coverage-file, --commit, --output, --chart
       - Load coverage.json
       - Get/create trend data
       - Calculate delta from previous
       - Update and save
       - Print summary with delta and chart

    Use update_coverage_trending.py as reference for existing patterns.
  </action>
  <verify>cd backend && python3 tests/scripts/coverage_trend_tracker.py --help</verify>
  <done>Script exists, accepts --commit and --chart flags, generates ASCII visualization</done>
