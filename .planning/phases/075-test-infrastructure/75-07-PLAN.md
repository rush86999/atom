### Task 3: Configure pytest.ini for E2E UI test discovery
**Files**:
- `backend/pytest.ini` (or create if not exists)

**Action**:
Create/update pytest.ini with:
```ini
[pytest]
testpaths = tests/e2e_ui/tests
python_files = test_*.py
python_classes = Test*
python_functions = test_*
markers =
    e2e: marks tests as end-to-end tests
    slow: marks tests as slow (deselect with -m "not slow")
addopts =
    -v
    --strict-markers
    --tb=short
    --disable-warnings
    -p no:warnings
asyncio_mode = auto
```

**Verify**:
```bash
grep -q "testpaths = tests/e2e_ui/tests" backend/pytest.ini
grep -q "asyncio_mode = auto" backend/pytest.ini
```

**Done**: When pytest.ini has E2E test configuration
