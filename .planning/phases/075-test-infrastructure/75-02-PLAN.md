## Success Criteria
1. authenticated_page fixture sets JWT token in localStorage (10-100x faster than UI login)

## Notes
- API-first login: Call /api/v1/auth/login with test credentials, get JWT, set in localStorage
- Bypasses slow UI login flow (typing email/password, clicking submit, waiting for redirect)
- data-testid attributes should be added to frontend components (deferring to Phase 76)
- Page Object Model separates test logic from UI interaction logic (maintainability)
