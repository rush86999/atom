# CI/CD Pattern from Research
1. On PR: Run quick smoke tests (50 users, 2 min) - don't block merges
2. On schedule (daily): Run full load tests (100 users, 5 min)
3. On main: Update baseline after successful tests
4. Artifact upload: HTML reports for manual inspection

# Workflow Triggers
- pull_request: paths include backend/core/**, backend/api/**
- schedule: cron '0 2 * * *' (daily at 2 AM UTC)
- workflow_dispatch: Manual trigger

# Job Steps
1. Checkout code
2. Set up Python
3. Install dependencies
4. Start application (uvicorn in background)
5. Run load tests
6. Compare to baseline
7. Upload report artifact
8. Update baseline (main branch only)
  </context>

  <name>Task 1: Create GitHub Actions load test workflow</name>
  <files>.github/workflows/load-test.yml</files>
  <action>
Create .github/workflows/load-test.yml with:

1. Workflow name: "Load Tests"

2. Triggers:
```yaml
on:
  pull_request:
    paths:
      - 'backend/core/**'
      - 'backend/api/**'
      - 'backend/tests/load/**'
  schedule:
    - cron: '0 2 * * *'  # Daily at 2 AM UTC
  workflow_dispatch:  # Manual trigger
```

3. Single job "load-test" running on ubuntu-large

4. Steps:
   - Checkout code (actions/checkout@v4)
   - Set up Python (actions/setup-python@v5, python 3.11)
   - Install dependencies (pip install -r requirements.txt -r requirements-testing.txt)
   - Start application (background uvicorn)
   - Wait for startup (sleep 10)
   - Run load tests (./run_load_tests.sh with PR-friendly params)
   - Compare to baseline (./compare_performance.py)
   - Upload report (actions/upload-artifact@v4)

5. Application startup:
```yaml
- name: Start application
  run: |
    cd backend
    python -m uvicorn main:app --host 0.0.0.0 --port 8000 &
    echo $! > uvicorn.pid
    sleep 10

- name: Shutdown application
  if: always()
  run: |
    cd backend
    if [ -f uvicorn.pid ]; then
      kill $(cat uvicorn.pid) || true
    fi
```

6. Load test parameters:
   - PR: 50 users, 5 spawn rate, 2 min duration (quick smoke test)
   - Schedule: 100 users, 10 spawn rate, 5 min duration

7. Baseline update (main branch only):
```yaml
- name: Update baseline
  if: github.ref == 'refs/heads/main'
  run: |
    cp backend/tests/load/reports/load_test_*.json backend/tests/load/reports/baseline.json
    git config user.name "github-actions[bot]"
    git add backend/tests/load/reports/baseline.json
    git commit -m "Update load test baseline"
    git push
```

Add appropriate permissions for git operations.
