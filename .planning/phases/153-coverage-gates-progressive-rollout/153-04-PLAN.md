   # Coverage Enforcement Guide (Phase 153)

   ## Progressive Rollout Strategy

   Atom uses a three-phase rollout to enforce coverage thresholds without blocking development:

   - **Phase 1 (70%)**: Baseline enforcement - Prevents coverage regression
   - **Phase 2 (75%)**: Interim target - Incremental improvement
   - **Phase 3 (80%)**: Final target - Production quality

   ## Platform-Specific Thresholds

   - **Backend**: 70% → 75% → 80% (diff-cover enforcement)
   - **Frontend**: 70% → 75% → 80% (Jest thresholds)
   - **Mobile**: 50% → 55% → 60% (jest-expo thresholds, lower due to React Native complexity)
   - **Desktop**: 40% → 45% → 50% (cargo-tarpaulin, lower due to Rust unsafe code)

   ## New Code Enforcement

   All new files require 80% coverage regardless of current phase.
   ```

   def track_bypass_usage(reason: str, pr_url: str, approvers: list[str]):
       """Track emergency bypass usage for audit trail."""
       bypass_log = Path(__file__).parent.parent / "coverage_reports" / "metrics" / "bypass_log.json"

       # Create bypass entry
       entry = {
           "timestamp": datetime.utcnow().isoformat(),
           "reason": reason,
           "pr_url": pr_url,
           "approvers": approvers,
           "phase": os.getenv("COVERAGE_PHASE", "phase_1"),
           "environment": os.getenv("ENVIRONMENT", "unknown")
       }

       # Load existing log
       if bypass_log.exists():
           with open(bypass_log) as f:
               log = json.load(f)
       else:
           log = {"bypasses": []}

       # Add entry
       log["bypasses"].append(entry)

       # Save log
       bypass_log.parent.mkdir(parents=True, exist_ok=True)
       with open(bypass_log, 'w') as f:
           json.dump(log, f, indent=2)

       return entry
   ```

3. Alert on excessive bypass:
   ```python
   def check_bypass_frequency():
       """Check if bypass usage exceeds threshold (>3 per month)."""
       bypass_log = Path(__file__).parent.parent / "coverage_reports" / "metrics" / "bypass_log.json"

       if not bypass_log.exists():
           return False

       with open(bypass_log) as f:
           log = json.load(f)

       # Count bypasses in last 30 days
       cutoff = datetime.utcnow().timestamp() - (30 * 24 * 60 * 60)
       recent_bypasses = [
           b for b in log["bypasses"]
           if datetime.fromisoformat(b["timestamp"]).timestamp() > cutoff
       ]

       if len(recent_bypasses) > 3:
           print("⚠️  WARNING: More than 3 emergency bypasses in last 30 days")
           print(f"   Recent bypasses: {len(recent_bypasses)}")
           print("   Consider: Investigating root causes, adjusting thresholds")
           return True

       return False
   ```
