<task type="auto">
  <name>Task 1: Create unified quality gate enforcement script</name>
  <files>backend/tests/scripts/ci_quality_gate.py</files>
  <action>
    Create backend/tests/scripts/ci_quality_gate.py that enforces all quality gates:

    1. Coverage gate:
       - Reads coverage.json for current coverage
       - Checks 80% line coverage minimum
       - Checks 70% branch coverage minimum
       - Fails with clear message if below threshold

    2. Pass rate gate:
       - Reads pytest output or test_health.json
       - Checks 98% pass rate minimum
       - Fails with failing test count if below threshold

    3. Regression gate:
       - Reads trending.json for baseline coverage
       - Checks coverage hasn't dropped >5% from baseline
       - Fails with regression amount if exceeded

    4. Flaky test gate:
       - Reads test_health.json for flaky tests
       - Warns if >5% of tests are flaky
       - Fails if >10% of tests are flaky

    Script structure:
    - check_coverage_gate() -> (passed, message)
    - check_pass_rate_gate() -> (passed, message)
    - check_regression_gate() -> (passed, message)
    - check_flaky_test_gate() -> (passed, message)
    - print_summary(results) -> formatted output
    - main() with --strict, --coverage-min, --pass-rate-min options

    Use json for parsing, argparse for CLI, sys.exit for return codes.
    Output format: "GATE_NAME: PASS/FAIL - Details"
  </action>
  <verify>
    python backend/tests/scripts/ci_quality_gate.py --help
  </verify>
  <done>
    Unified quality gate script enforces coverage, pass rate, regression, and flaky test thresholds. Provides clear pass/fail output with remediation steps.
  </done>
