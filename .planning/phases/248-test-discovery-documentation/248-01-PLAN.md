## Known Pattern

The syntax error pattern is:
```python
# Start audit logging
audit_ctx = log_integration_attempt("service_name", "method_name", locals())
try:
    # Check circuit breaker
    if not await circuit_breaker.is_enabled("service_name"):
        logger.warning(f"Circuit breaker is open for service_name")
        log_integration_complete(audit_ctx, error=Exception("Circuit breaker open"))
        raise HTTPException(
            status_code=503,
            detail=f"Service_name integration temporarily disabled"
        )

    # Check rate limiter
    is_limited, remaining = await rate_limiter.is_rate_limited("service_name")
    if is_limited:
        logger.warning(f"Rate limit exceeded for service_name")
        log_integration_complete(audit_ctx, error=Exception("Rate limit exceeded"))
        raise HTTPException(
            status_code=429,
            detail=f"Rate limit exceeded for service_name"
        )

# Missing: except or finally block here
```

The fix is to add:
```python
except HTTPException:
    raise
except Exception as e:
    log_integration_complete(audit_ctx, error=e)
    raise
```
