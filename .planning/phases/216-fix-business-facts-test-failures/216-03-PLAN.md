    **File: .planning/phases/216-fix-business-facts-test-failures/216-PATTERN-DOC.md**

    Content should include:

    1. **Mock Patching Pattern**
       - Always patch where the service is imported, not where it's defined
       - Pattern: `patch('api.module.routes.ServiceName')` not `patch('core.service_module.ServiceName')`
       - Example from this phase:
         ```python
         # Route file: api/admin/business_facts_routes.py
         from core.agent_world_model import WorldModelService

         # Test file patch location:
         patch('api.admin.business_facts_routes.WorldModelService', ...)
         ```

    2. **Error Response Assertion Pattern**
       - BaseAPIRouter.error_response() returns structured dict
       - Access nested message: `detail["error"]["message"]`
       - Example:
         ```python
         detail = response.json()["detail"]
         message = detail["error"]["message"]
         assert "not found" in message.lower()
         ```

    3. **Service Mock Fixture Pattern**
       - AsyncMock for async service methods
       - Configure return values in fixture, not in tests
       - Example from test file:
         ```python
         @pytest.fixture
         def mock_world_model_service(sample_business_fact):
             mock = AsyncMock()
             mock.get_fact_by_id.return_value = sample_business_fact
             mock.list_all_facts.return_value = [sample_business_fact]
             return mock
         ```

    4. **S3/R2 Storage Mock Pattern**
       - Mock upload_file, check_exists methods
       - Return deterministic S3 URIs
       - Example:
         ```python
         @pytest.fixture
         def mock_storage_service():
             mock = MagicMock()
             mock.upload_file.return_value = "s3://bucket/key"
             mock.check_exists.return_value = True
             return mock
         ```

    Include before/after examples from the actual test fixes.

    ```python

    from core.agent_world_model import WorldModelService

    # Test file patches at import location:

    ```

    **Error Response Assertions:**
    BaseAPIRouter.error_response() returns structured errors. Access nested message:

    ```python
    detail = response.json()["detail"]
    assert "not found" in detail["error"]["message"].lower()
    ```

    **Error Response Structure:**
    ```json
    {
      "success": false,
      "error": {
        "code": "NOT_FOUND",
        "message": "Resource not found",
        "timestamp": "2026-03-20T...",
        "details": {}
      }
    }
    ```

    **Service Mock Fixtures:**
    Use AsyncMock for async services, configure returns in fixture:

    ```python
    @pytest.fixture
    def mock_service():
        mock = AsyncMock()
        mock.get_method.return_value = expected_value
        return mock

    # Usage in test:
    with patch('api.routes.ServiceName', return_value=mock_service):
        response = client.get("/endpoint")
    ```
    ```
