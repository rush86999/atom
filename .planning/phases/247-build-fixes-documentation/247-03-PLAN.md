       # Atom Build Guide

       This guide explains how to build Atom from source.

       ## Prerequisites

       - Node.js 20.x (verify with `node --version`)
       - Python 3.11+ (verify with `python3 --version`)
       - npm 10.x (verify with `npm --version`)

       ## Frontend Build

       Location: `frontend-nextjs/`

       ### Development Build
       ```bash
       cd frontend-nextjs
       npm install
       npm run dev
       ```

       ### Production Build
       ```bash
       cd frontend-nextjs
       npm install
       npm run build
       ```

       The production build creates `.next/` directory with optimized assets.

       ### Troubleshooting Frontend Build

       **SWC Binary Error:**
       ```
       Failed to load SWC binary for darwin/x64
       ```
       Solution: `npm install @next/swc-darwin-x64@[next-version]`

       **Version Mismatch:**
       ```
       Mismatching @next/swc version
       ```
       Solution: Ensure @next/swc packages match Next.js version exactly

       **Minification Error:**
       ```
       ReferenceError: [truncated variable] is not defined
       ```
       Solution: This is a SWC bug. Try clearing cache: `rm -rf .next node_modules/.cache`

       ## Backend Build

       Location: `backend/`

       ### Development Setup
       ```bash
       cd backend
       pip install -e .
       ```

       ### Production Build
       ```bash
       cd backend
       python -m build
       ```

       This creates `dist/` directory with wheel and tarball.

       ### Verification
       ```bash
       # Check syntax
       python3 -m py_compile backend/**/*.py

       # Collect tests
       cd backend
       python3 -m pytest --collect-only
       ```

       ### Troubleshooting Backend Build

       **Syntax Error:**
       ```
       SyntaxError: expected 'except' or 'finally' block
       ```
       Solution: Check for unmatched try-except blocks in Python files

       **Import Error:**
       ```
       ModuleNotFoundError: No module named 'core'
       ```
       Solution: Install package in editable mode: `pip install -e .`

       ## Full Build (Both)

       To build both frontend and backend:

       ```bash
       # Frontend
       cd frontend-nextjs && npm install && npm run build && cd ..

       # Backend
       cd backend && python -m build && cd ..

       echo "Build complete!"
       ```

       ## Continuous Integration

       The CI/CD pipeline automatically builds on:
       - Every push to main branch
       - Pull requests
       - Manual trigger via GitHub Actions

       See `.github/workflows/` for CI configuration.

       ## Build Times

       - Frontend: ~10-15 minutes (first build), ~2-5 minutes (incremental)
       - Backend: ~1-2 minutes

       ## Environment Variables

       Required for builds:
       - Frontend: See `frontend-nextjs/.env.example`
       - Backend: See `backend/.env.example`

       ## Next Steps

       After building:
       - Run tests: See TESTING.md
       - Run locally: See DEVELOPMENT.md
       - Deploy: See DEPLOYMENT.md
       ```
