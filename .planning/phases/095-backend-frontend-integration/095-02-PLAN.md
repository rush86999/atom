    Create Python script with the following structure:

    1. **Imports**: argparse, json, sys, pathlib.Path, typing (Dict, Any, Optional)

    2. **load_pytest_coverage(path: Path) -> Dict[str, Any]** function:
       - Load pytest coverage.json from backend/tests/coverage_reports/metrics/coverage.json
       - Extract totals: percent_covered, covered_lines, num_statements
       - Return dict with platform='python', coverage_pct, covered, total metrics
       - Handle FileNotFoundError by returning platform='python', coverage_pct=0.0, error='file not found'

    3. **load_jest_coverage(path: Path) -> Dict[str, Any]** function:
       - Load Jest coverage-final.json from frontend-nextjs/coverage/coverage-final.json
       - Parse Jest format: total lines covered/total lines (compute percentage)
       - Return dict with platform='javascript', coverage_pct, covered, total metrics
       - Handle FileNotFoundError by returning platform='javascript', coverage_pct=0.0, error='file not found'

    4. **aggregate_coverage(pytest_path: Path, jest_path: Path) -> Dict[str, Any]** function:
       - Call both load functions
       - Compute overall coverage: (covered_backend + covered_frontend) / (total_backend + total_frontend)
       - Return dict with:
         - platforms: {python: {...}, javascript: {...}}
         - overall: {coverage_pct, covered, total}
         - timestamp: ISO format datetime

    5. **generate_report(aggregate_data: Dict[str, Any], format: str)** function:
       - If format='json': print JSON with --indent 2
       - If format='text': print human-readable table with platform breakdowns
       - If format='markdown': print markdown table for PR comments

    6. **main()** function with argparse:
       - --pytest-coverage default=backend/tests/coverage_reports/metrics/coverage.json
       - --jest-coverage default=frontend-nextjs/coverage/coverage-final.json
       - --output default=backend/tests/scripts/coverage_reports/unified/coverage.json
       - --format choices=[json, text, markdown] default=text
       - Parse paths, call aggregate_coverage, save JSON output, print report
