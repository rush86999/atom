<task type="auto">
  <name>Task 3: Run 3 consecutive test passes for stability verification</name>
  <files>backend</files>
  <action>
    1. Create a loop script to run 3 times:
       ```bash
       for i in 1 2 3; do
         echo "=== Run $i ==="
         time pytest -n auto --dist=loadscope -q 2>&1 | tee /tmp/run_${i}.log
         echo "Exit code: $?"
       done
       ```
    2. Run the script and capture results
    3. Document:
       - Pass/fail counts for each run
       - Any tests that fail intermittently (flaky)
       - Whether failures are consistent (real bugs) or intermittent (flaky)
    4. Calculate flaky test rate: (flaky tests / total tests) * 100
    5. Target: 100% pass rate, <=8% flaky rate

    If all 3 runs pass with same results, tests are stable.
  </action>
  <verify>for i in 1 2 3; do echo "Run $i:"; grep -E "passed|failed" /tmp/run_${i}.log 2>/dev/null || echo "No log"; done</verify>
  <done>
    - 3 consecutive runs executed
    - Results documented for each run
    - Flaky tests identified (if any)
    - Pass rate calculated
  </done>
</task>
