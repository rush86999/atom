<objective>
Replace hardcoded environment variable usage with monkeypatch fixture for proper test isolation.

This plan addresses environment variable pollution, a common cause of flaky tests. Direct os.getenv() calls and os.environ modifications in tests cause tests to fail when run in different orders or in parallel. The monkeypatch fixture provides automatic cleanup.

Purpose: Eliminate environment-dependent test failures
Output: Tests using monkeypatch for all environment variable isolation
</objective>
