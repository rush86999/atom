  **backend/core/budget_enforcement_service.py** (180 lines):
  - Import Decimal, to_decimal, round_money from decimal_utils
  - Import Session from sqlalchemy.orm
  - Import Project, BudgetStatus from service_delivery.models

  Custom exceptions:
  ```python
  class BudgetError(Exception):
      """Base exception for budget operations"""
      pass

  class InsufficientBudgetError(BudgetError):
      """Raised when spend would exceed budget limit"""
      def __init__(self, requested: Decimal, remaining: Decimal, budget_id: str = None):
          self.requested = requested
          self.remaining = remaining
          self.budget_id = budget_id
          super().__init__(f"Requested {requested}, only {remaining} remaining")

  class BudgetNotFoundError(BudgetError):
      """Raised when budget doesn't exist"""
      pass
  ```

  BudgetEnforcementService class:
  1. `__init__(self, db: Session)`: Initialize with database session

  2. `check_budget(self, project_id: str, amount: Union[Decimal, str, float]) -> Dict[str, Any]`:
     - Query Project by project_id
     - Raise BudgetNotFoundError if not found
     - Convert amount to Decimal using to_decimal()
     - Calculate: remaining = budget_amount - actual_burn
     - Return: {"allowed": bool, "remaining": Decimal, "budget_status": str, "utilization_pct": Decimal}

  3. `approve_spend(self, project_id: str, amount: Union[Decimal, str, float], description: str = None) -> Dict[str, Any]`:
     - Check budget using check_budget()
     - Raise InsufficientBudgetError if not allowed
     - Update Project.actual_burn atomically
     - Recalculate budget_status (on_track <80%, at_risk 80-99%, over_budget >=100%)
     - Return approval result with new remaining

  4. `record_spend(self, project_id: str, amount: Union[Decimal, str, float], category: str, description: str = None) -> Dict[str, Any]`:
     - Call approve_spend() (raises if insufficient)
     - Create Transaction record linked to project
     - Return transaction details

  5. `get_budget_status(self, project_id: str) -> Dict[str, Any]`:
     - Return full budget status (amount, burn, remaining, status, utilization)

  IMPORTANT: All database operations must use explicit transactions (db.begin()/commit()/rollback()). No check-then-act outside transaction.
