<task type="auto">
  <name>Task 2: Create SQLite quarantine database</name>
  <files>backend/tests/scripts/flaky_test_tracker.py</files>
  <action>
Create new file flaky_test_tracker.py with:

1. FlakyTestTracker class with:
   - __init__(db_path: Path): Create connection, call _create_schema()
   - _create_schema(): CREATE TABLE flaky_tests with columns:
     * id INTEGER PRIMARY KEY AUTOINCREMENT
     * test_path TEXT NOT NULL
     * platform TEXT NOT NULL (backend/frontend/mobile/desktop)
     * first_detected TEXT NOT NULL (ISO timestamp)
     * last_detected TEXT NOT NULL (ISO timestamp)
     * total_runs INTEGER NOT NULL DEFAULT 0
     * failure_count INTEGER NOT NULL DEFAULT 0
     * flaky_rate REAL NOT NULL DEFAULT 0.0
     * classification TEXT NOT NULL (stable/flaky/broken)
     * failure_history TEXT NOT NULL (JSON array of timestamps)
     * quarantine_reason TEXT
     * issue_url TEXT
     * created_at TEXT NOT NULL DEFAULT (datetime('now'))
     * updated_at TEXT NOT NULL DEFAULT (datetime('now'))
   - Indexes on (test_path, platform) and flaky_rate DESC

2. record_flaky_test() method:
   - Check if test_path + platform exists
   - If exists: UPDATE with merged failure_history, updated timestamp
   - If new: INSERT with all fields
   - Return test_id

3. get_quarantined_tests(platform: Optional[str]) method:
   - SELECT * WHERE classification = 'flaky'
   - Optional platform filter
   - Return list of dicts

4. get_test_reliability_score(test_path, platform) method:
   - Return 1.0 - flaky_rate (0.0 to 1.0)
   - Return 1.0 if test not found (no failures = perfect)

Reference RESEARCH.md Pattern 2 for schema design.
  </action>
