  <action>
    Create jest-axe fixtures for accessibility testing:

    1. **Install jest-axe** (via Bash):
       ```bash
       cd frontend-nextjs
       npm install --save-dev jest-axe @playwright/test
       ```

    2. **Create axe_fixtures.py** with:
       - **axe_page fixture**: Playwright page with axe-core injected
       - **axe_run helper**: Run axe-core and return violations
       - **axe_check_violations helper**: Assert no violations, print details if found
       - **accessible_page fixture**: Authenticated page ready for a11y testing

    3. **Implementation**:
       ```python
       import pytest
       from playwright.sync_api import Page

       @pytest.fixture
       def axe_page(browser):
           context = browser.new_context()
           page = context.new_page()
           # Inject axe-core via CDN or local file
           page.goto("about:blank")
           page.evaluate("""() => {
             const script = document.createElement('script');
             script.src = 'https://cdnjs.cloudflare.com/ajax/libs/axe-core/4.8.2/axe.min.js';
             document.head.appendChild(script);
           }""")
           page.wait_for_load_state("networkidle")
           yield page
           context.close()

       def axe_run(page):
           """Run axe-core and return violations"""
           return page.evaluate("""() => {
             return axe.run().then(results => results.violations);
           }""")

       def axe_check_violations(violations, page_name="Page"):
           """Assert no violations, print details if found"""
           if violations:
               print(f"\n❌ Accessibility violations found on {page_name}:")
               for v in violations:
                   print(f"  - {v['id']}: {v['description']}")
                   print(f"    Impact: {v['impact']}")
                   print(f"    Nodes: {len(v['nodes'])} affected")
               pytest.fail(f"Found {len(violations)} accessibility violations")
           else:
               print(f"✅ No accessibility violations on {page_name}")
       ```

    Use axe-core 4.8.2 or later (latest stable).
    Inject axe-core via CDN for simplicity (or bundle locally).
    Use pytest fixtures with proper setup/teardown.
  </action>
