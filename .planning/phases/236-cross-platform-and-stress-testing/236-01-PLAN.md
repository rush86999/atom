    Create k6 setup and configuration file `backend/tests/load/k6_setup.js`:

    1. **Install k6** (via Bash):
       ```bash
       # Check if k6 is installed
       if ! command -v k6 &> /dev/null; then
         # Install via homebrew (macOS)
         if [[ "$OSTYPE" == "darwin"* ]]; then
           brew install k6
         # Or download binary for Linux
         else
           curl https://github.com/grafana/k6/releases/download/v0.51.0/k6-v0.51.0-linux-amd64.tar.gz -L | tar xvz
           sudo mv k6-v0.51.0-linux-amd64/k6 /usr/local/bin/
         fi
       fi
       ```

    2. **Create k6_setup.js** with:
       - Base URL configuration: `const BASE_URL = __ENV.API_URL || 'http://localhost:8000'`

       - Helper functions for authentication (get valid token via /api/auth/login)
       - Helper functions for random user selection
       - Helper functions for agent execution
       - Export options object with threshold configuration

    DO NOT install k6 via npm (it's a standalone binary, not an npm package).
