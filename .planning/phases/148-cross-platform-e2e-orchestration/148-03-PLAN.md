    3. **Local Setup** section:
       - Install dependencies: `pip install pytest-playwright playwright`
       - Install browsers: `playwright install chromium`
       - Start services: Backend on port 8001, frontend on port 3001

    4. **Writing Tests** section:
       - Use Page fixture from playwright.sync_api
       - Use expect() assertions for auto-waiting
       - Add @pytest.mark.e2e marker for CI/CD
       - Cleanup after test with autouse fixtures

    5. **Common Issues** section:
       - Port 8001/3001 in use: Kill existing processes
       - Element timeout: Use page.wait_for_selector()
       - Test isolation: Use unique IDs, cleanup fixtures
