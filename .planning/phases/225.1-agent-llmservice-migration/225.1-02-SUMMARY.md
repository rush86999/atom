## Methods Added

### 1. analyze_query_complexity(prompt, task_type=None)

**Purpose:** Analyze query complexity to determine optimal provider routing

**Used by:** generic_agent.py (line 222) for TRACE Framework metrics

**Returns:** QueryComplexity enum (SIMPLE, MODERATE, COMPLEX, ADVANCED)

**Example:**
```python
service = LLMService()
complexity = service.analyze_query_complexity("Write a Python REST API")
print(complexity.value)  # 'complex' or 'advanced'
```

**Delegation:** `return self.handler.analyze_query_complexity(prompt, task_type)`

### 2. get_available_providers()

**Purpose:** Get list of providers with valid API keys

**Returns:** List[str] of provider IDs (e.g., ["openai", "anthropic", "deepseek"])

**Example:**
```python
service = LLMService()
providers = service.get_available_providers()
print(providers)  # ['openai', 'deepseek', ...]
```

**Delegation:** `return self.handler.get_available_providers()`

### 3. get_context_window(model_name)

**Purpose:** Get context window size for a model from dynamic pricing data

**Returns:** int (context window size in tokens, e.g., 128000, 200000)

**Example:**
```python
service = LLMService()
context = service.get_context_window("gpt-4o")
print(context)  # 128000
```

**Delegation:** `return self.handler.get_context_window(model_name)`

### 4. truncate_to_context(text, model_name, reserve_tokens=1000)

**Purpose:** Truncate text to fit within model's context window

**Returns:** str (truncated text with truncation indicator)

**Example:**
```python
service = LLMService()
short = service.truncate_to_context(long_text, "gpt-4o", reserve_tokens=2000)
```

**Delegation:** `return self.handler.truncate_to_context(text, model_name, reserve_tokens)`
