Create agent execution property tests validating core invariants (idempotence, termination, determinism).

**Purpose:** PROP-01 requires 50+ new property tests for critical paths. Agent execution is the most critical path - every agent interaction flows through execution. Invariant violations here cause data corruption, infinite loops, or non-deterministic behavior.

**Output:** 3 new test files with 10-12 property tests covering execution idempotence, graceful termination, and determinism invariants.

**Why this matters:** Property tests find edge cases example-based tests miss. Example: "Agent execution terminates" tests one agent. Property test: "ALL agents terminate for ALL valid inputs" tests thousands of auto-generated cases (empty params, huge payloads, null values, unicode, etc.).
