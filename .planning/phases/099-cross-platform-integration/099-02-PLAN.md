<objective>
Spike Detox E2E test infrastructure setup for React Native mobile app with Expo 50. Verify feasibility of Detox integration (dev client requirement, build complexity) and establish baseline for mobile E2E tests that mirror web cross-platform tests from Plan 01.

Purpose: Detox is the industry standard for React Native E2E testing (grey-box architecture, 10x faster than Appium), but Expo integration requires verification. This plan creates a minimal working Detox setup to validate feasibility before committing to full implementation in Plan 04. If Detox proves too complex (requires ejecting or dev client), we can fallback to API-level testing or defer mobile E2E to post-v4.0.

Output: Working Detox configuration with Expo 50, prototype test passing, feasibility assessment, and documentation of next steps.
</objective>
