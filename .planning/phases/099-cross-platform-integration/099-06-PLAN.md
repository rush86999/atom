  <files>frontend-nextjs/package.json, frontend-nextjs/lighthouserc.json</files>
  <action>
Install Lighthouse CI and create configuration with performance budgets:

1. Add Lighthouse CI to devDependencies:
   ```bash
   cd frontend-nextjs
   npm install --save-dev @lhci/cli puppeteer
   ```

2. Update `package.json` scripts:
   ```json
   "scripts": {
     "lighthouse:ci": "lhci autorun",
     "lighthouse:collect": "lhci collect",
     "lighthouse:assert": "lhci assert"
   }
   ```

3. Create `frontend-nextjs/lighthouserc.json`:
   ```json
   {
     "ci": {
       "collect": {
         "url": [
           "http://localhost:3001/dashboard",
           "http://localhost:3001/agent/test-agent",
           "http://localhost:3001/canvas/test-canvas"
         ],
         "numberOfRuns": 3,
         "settings": {
           "preset": "desktop",
           "throttling": {
             "rttMs": 40,
             "throughputKbps": 10 * 1024,
             "cpuSlowdownMultiplier": 1,
             "requestLatencyMs": 0,
             "downloadThroughputKbps": 0,
             "uploadThroughputKbps": 0
           }
         }
       },
       "upload": {
         "target": "temporary-public-storage"
       },
       "assert": {
         "assertions": {
           "categories:performance": ["error", { "minScore": 0.9 }],
           "categories:accessibility": ["warn", { "minScore": 0.8 }],
           "categories:best-practices": ["warn", { "minScore": 0.9 }],
           "categories:seo": ["off"],
           "first-contentful-paint": ["error", { "maxNumericValue": 2000 }],
           "interactive": ["error", { "maxNumericValue": 5000 }],
           "cumulative-layout-shift": ["warn", { "maxNumericValue": 0.1 }],
           "total-blocking-time": ["error", { "maxNumericValue": 300 }]
         }
       }
     }
   }
   ```

4. Document rationale for budgets:
   - Performance score 90+ (A grade) prevents slow pages
   - FCP < 2s ensures fast initial render
   - TTI < 5s ensures interactivity within 5 seconds
   - CLS < 0.1 prevents layout shift issues
   - TBT < 300ms keeps main thread responsive

3. Create `backend/tests/e2e_ui/tests/visual/test_visual_regression.py`:
   ```python
   from tests.e2e_ui.pages.page_objects import DashboardPage, AgentChatPage, CanvasPage
   import pytest

   class TestVisualRegression:
       \"\"\"Visual regression tests for critical pages.\"\"\"

       def test_dashboard_visual(self, browser, authenticated_user):
           \"\"\"Verify dashboard page has no visual regressions.\"\"\"
           dashboard = DashboardPage(browser.new_page())
           dashboard.navigate()
           percy_snapshot(browser.page, name="Dashboard Page")

       def test_agent_chat_visual(self, browser, authenticated_user):
           \"\"\"Verify agent chat page has no visual regressions.\"\"\"
           chat = AgentChatPage(browser.new_page())
           chat.navigate()
           chat.send_message("Test message for snapshot")
           percy_snapshot(browser.page, name="Agent Chat Page")

       def test_canvas_sheets_visual(self, browser, authenticated_user):
           \"\"\"Verify canvas sheets presentation has no visual regressions.\"\"\"
           canvas = CanvasPage(browser.new_page())
           canvas.navigate()
           canvas.present_canvas(type="sheets", data={"rows": 5, "columns": 3})
           percy_snapshot(browser.page, name="Canvas Presentation - Sheets")
   ```

  <name>Task 5: Create Lighthouse CI workflow and bundle size tracking</name>
  <files>.github/workflows/lighthouse-ci.yml</files>
  <action>
Create CI workflow for Lighthouse performance tests and configure bundle size tracking:

1. Create `.github/workflows/lighthouse-ci.yml`:
   ```yaml
   name: Lighthouse CI

   on:
     pull_request:
       branches: [main]
     push:
       branches: [main]
     workflow_dispatch:

   jobs:
     lighthouse:
       runs-on: ubuntu-latest
       timeout-minutes: 15

       steps:
         - name: Checkout code
           uses: actions/checkout@v4

         - name: Setup Node.js
           uses: actions/setup-node@v4
           with:
             node-version: '20'

         - name: Install dependencies
           working-directory: ./frontend-nextjs
           run: npm ci

         - name: Build frontend
           working-directory: ./frontend-nextjs
           run: npm run build

         - name: Start server
           working-directory: ./frontend-nextjs
           run: |
             npm start &
             npx wait-on http://localhost:3001

         - name: Run Lighthouse CI
           working-directory: ./frontend-nextjs
           run: npm run lighthouse:ci
           env:
             LHCI_GITHUB_APP_TOKEN: ${{ secrets.LHCI_GITHUB_APP_TOKEN }}

         - name: Upload Lighthouse results
           if: always()
           uses: actions/upload-artifact@v4
           with:
             name: lighthouse-results
             path: frontend-nextjs/.lighthouseci/
             retention-days: 30

         - name: Comment PR with results
           if: github.event_name == 'pull_request'
           uses: actions/github-script@v7
           with:
             script: |
               const fs = require('fs');
               const resultsPath = 'frontend-nextjs/.lighthouseci/lhci-report.html';
               if (fs.existsSync(resultsPath)) {
                 // Post summary to PR
                 console.log('Lighthouse results available in artifacts');
               }
   ```

2. Add bundle size tracking to `frontend-nextjs/next.config.js`:
   ```javascript
   // Add bundle size tracking with next-bundle-analysis
   const withBundleAnalyzer = require('@next/bundle-analyzer')({
     enabled: process.env.ANALYZE === 'true',
   });

   module.exports = withBundleAnalyzer({
     // Existing config...
   });
   ```

3. Add bundle analyzer to devDependencies:
   ```bash
   cd frontend-nextjs
   npm install --save-dev @next/bundle-analyzer
   ```

4. Update package.json scripts:

   ```

       },

         "limit": "200 kB"
       }
     ]
   }
   ```
