# Canvas Accessibility Context
- AI accessibility: Hidden trees expose state as JSON for screen readers
- Hidden tree attributes: role='log', aria-live='polite' or 'assertive'
- Hidden trees allow AI agents to read canvas state without OCR
- Format: <div role='log' aria-live='polite' style='visually-hidden'>{JSON state}</div>
- Enables screen reader announcements of canvas changes
- Each canvas component should have its own accessibility tree
- State JSON must be properly escaped to prevent XSS
</context>
