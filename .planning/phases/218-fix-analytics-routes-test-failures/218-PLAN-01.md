# Plan 218-01: Fix Analytics Routes Test Fixture Errors

**Goal:** Fix 10 failing tests in test_analytics_routes.py by replacing non-existent `db` fixture with `db_session`.

## Problem

All tests in `tests/api/test_analytics_routes.py` fail with:
```
ERROR at setup
fixture 'db' not found
available fixtures: db_session, test_db, mock_db_session, ...
```

The `mock_user` fixture (line 42) and all test functions request a `db` parameter that doesn't exist.

## Root Cause

The tests were written with a `db` fixture, but this fixture doesn't exist in the conftest.py.
The correct fixture name is `db_session` (which is in the available fixtures list).

## Solution

Replace all instances of `db` with `db_session` in:
1. The `mock_user` fixture definition (line 42)
2. All test function signatures that use `db: Session`
